Sincronização MySQL para Excel

Versão: 1.1 (Dinâmica)

Este projeto oferece um conjunto de scripts Python poderosos para facilitar a sincronização bidirecional e dinâmica entre um banco de dados MySQL e arquivos Excel. Ele permite que você extraia dados de qualquer tabela do seu MySQL para uma planilha Excel e, inversamente, importe dados de qualquer planilha Excel para uma nova tabela no seu banco de dados.

Com estes scripts, você pode automatizar a gestão de dados, tornando o processo de importação e exportação flexível e eficiente, sem a necessidade de modificações manuais no código para cada nova tabela.

Tecnologias Utilizadas

O projeto é desenvolvido em Python, utilizando bibliotecas essenciais para manipulação de dados e interação com bancos de dados e arquivos Excel:

  • pandas: Para leitura, manipulação e exportação de dados em formato tabular (DataFrames).
  • mysql-connector-python: Para estabelecer conexão e interagir com o banco de dados MySQL.
  • openpyxl: Para manipulação avançada de arquivos Excel, como ajuste automático de largura de colunas e formatação.
  • os: Para operações de sistema de arquivos, como gerenciamento de caminhos.

Melhorias Futuras

  • Interface Gráfica (GUI): Implementar uma interface de usuário intuitiva para facilitar a seleção de arquivos, tabelas e configurações de importação/exportação, eliminando a necessidade de interação direta com o código.
  • Mapeamento de Tipos Avançado: Adicionar opções de configuração para lidar com tipos de dados complexos ou específicos, permitindo maior controle sobre o esquema da tabela MySQL gerada a partir do Excel.
  • Tratamento de Conflitos (Upsert): Adicionar lógica para lidar com a atualização de registros existentes durante a importação (UPSERT), em vez de apenas inserção.
  • Log de Operações: Implementar um sistema de log detalhado para registrar o sucesso ou falha das operações, facilitando a depuração e o monitoramento.
