Sincronização MySQL & Excel

Versão: 1.0 (GUI Interativa)

Este projeto oferece um conjunto de scripts Python poderosos que facilitam a sincronização bidirecional e dinâmica entre um banco de dados MySQL e arquivos Excel.

Ele permite que você:

  • Extraia dados de qualquer tabela do seu MySQL para uma planilha Excel.
  • Importe dados de qualquer planilha Excel para uma nova tabela no seu banco de dados.
  • Tudo isso através de uma Interface Gráfica de Usuário (GUI) moderna e intuitiva.

Com este aplicativo, você pode automatizar a gestão de dados, tornando o processo de importação e exportação flexível e eficiente, sem a necessidade de modificações manuais no código para cada nova tabela ou operação.

Tecnologias Utilizadas

O projeto é desenvolvido em Python, utilizando bibliotecas essenciais para manipulação de dados, criação da interface gráfica e interação com bancos de dados e arquivos Excel:

  • `CustomTkinter`: Para a criação da interface gráfica moderna e interativa.
  • `pandas`: Para leitura, manipulação e exportação eficiente de dados tabulares.
  • `mysql-connector-python`: Para estabelecer conexão e interagir com o banco de dados MySQL.
  • `openpyxl`: Para manipulação e estilização avançada de arquivos Excel.
  • `threading`: Para manter a interface gráfica responsiva durante operações de dados demoradas.
  • `json`: Para persistência e carregamento das configurações do banco de dados.
  • `os`: Para operações de sistema de arquivos e gerenciamento de caminhos.

Melhorias Futuras

  • Mapeamento de Colunas Avançado: Adicionar opções na GUI para mapear e transformar colunas durante a importação, garantindo maior controle sobre o esquema da tabela MySQL.
  • Mais Modos de Importação: Implementar opções adicionais como `UPSERT` (Atualizar/Inserir) ou `UPDATE ONLY` para cenários de sincronização de dados mais complexos.
  • Gerenciamento de Erros Aprimorado: Incluir um sistema de log detalhado para arquivo e notificações mais específicas na GUI para facilitar a depuração e o monitoramento.
  • Suporte a Outros Bancos de Dados: Expandir a compatibilidade para incluir outros sistemas de gerenciamento de banco de dados, como PostgreSQL ou SQLite.
