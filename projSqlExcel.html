Projeto Conversor SQL para Excel

Versão: 1.0

Este projeto contém dois scripts que facilitam a integração entre um banco de dados MySQL e um arquivo Excel, permitindo a extração de dados, a manipulação e a reintegração desses dados de volta ao banco.

Foram montados dois scripts atuais, sendo um para o envio da tabela Excel para o banco de dados e um de extração do banco de dados para a tabela Excel

Técnologias utilizadas

Foi utilizada de linguagem principal o Python com a integração de bibliotecas como:

  • Pandas
  • OS
  • MySqlConnector

Melhorias a serem implementadas

  • Interface para a melhor utilização
  • Modificar dentro da interface como quer as tabelas e/ou banco
